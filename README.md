# Lecture Summariser

A webapp that takes lecture PDFs or slides and returns:
- A concise summary
- Key concepts extracted
- A downloadable revision card

## Stack
- Frontend: React
- Backend: FastAPI (Python)
- ML: HuggingFace BART (facebook/bart-large-cnn)
- Storage: Supabase
- File parsing: PyMuPDF (PDFs), python-pptx (slides)
